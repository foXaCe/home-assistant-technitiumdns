# TechnitiumDNS

Integrate your [Technitium DNS Server](https://technitium.com/dns/) with Home Assistant:
DNS statistics sensors, ad-blocking switch with temporary-disable buttons, DHCP device
trackers, and smart activity analysis.

## Features

- DNS statistics sensors (queries, blocked, clients, top domains, cache, and more)
- Ad-blocking switch with 5/10/30/60 min and 1 day temporary-disable buttons
- DHCP device trackers with per-device diagnostic sensors
- Smart activity analysis to distinguish real usage from background traffic

## Configuration

Settings → Devices & Services → Add Integration → **TechnitiumDNS**.

You will need the API URL and a dedicated API token for your Technitium DNS Server.
See the README for the recommended user permissions.
